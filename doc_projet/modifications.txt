
18. Implémentation des Modules Admin (Projets, RH, Pubs)
--------------------------------------------------------
Date: 2026-02-17
Description: Création des interfaces d'administration restantes pour la gestion des projets, du personnel et des ressources (pubs/ouvrages).

Détails des ajouts :
1. Interface "Projets" (/admin/projets) :
   - Liste des projets avec barre de progression et statut.
   - Formulaire complet de création/édition :
     - Infos générales, Budget, Objectifs (liste dynamique).
     - Assignation d'équipe (Sélection de membres).
     - Gestion des étapes (Milestones) avec dates et état (Completed/Pending).

2. Interface "Personnel (RH)" (/admin/personnel) :
   - Système d'onglets pour gérer 3 types de profils :
     - Utilisateurs (Staff) : Filtrage par Rôle (Super Admin, Comptable, etc.).
     - Bénévoles : Filtrage par Compétence.
     - Partenaires : Filtrage par Domaine et Pays.

3. Interface "Pubs & Ouvrages" (/admin/pubs-ouvrages) :
   - Onglet Publicités : Gestion des campagnes (Date début/fin, Type Média, Stats de clics).
   - Onglet Ouvrages : Gestion des livres (PDF, Prix, Couverture).

4. Mises à jour techniques :
   - Extension des types TypeScript (Project, User, Volunteer, Partner, Advertisement, Book).
   - Mise à jour de la Sidebar Admin avec les nouveaux liens et icônes appropriées.

État : TERMINÉ (Frontend Mocké)
Prochaine étape : Backend Integration & Auth.
